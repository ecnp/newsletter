Autenticación

SDK para el manejo de autenticación y autorización de las aplicaciones utilizando Grant Type Password de OAuth 2.0
Storage

SDK para el manejo seguro de los gestores de almacenamiento de navegador (Session Storage y Local Storage)
Encriptación

SDK para el manejo de encriptación simétrica y asimétrica, bien sea en browser o para encriptación de tramas en tráfico
Tracking

SDK para el manejo de logs de acuerdo al nivel de incidencias en las aplicaciones y envío de estas hacia API's REST de soporte
Layout
